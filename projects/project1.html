Personalized Book Recommendation System

This project delivers a personalized book recommendation experience, leveraging data science and machine learning to provide suggestions based on my reading history and preferences.

Project Goals

  • Develop an accurate and user-friendly book recommendation engine.
  • Enable discovery of new books based on user reading history and preferences.
  • Create an interactive website to showcase the recommendations and allow user interaction.

Data and Approach

The dataset includes detailed information on each book, such as title, author, release date, editor, rating, summary, genre, and image ( more information about data collection ). The recommendation engine uses a (hybrid) content-based filtering approach, comparing books and user books based on genre, author, ratings and summaries to recommend similar books.

I converted book summaries into embeddings using HuggingFace's French models. Additionally, genres and authors were one-hot encoded, while ratings were normalized. The core concept is to calculate cosine similarity for each attribute (genre, content, and author) between the books I've read and the entire collection, culminating in a unified similarity matrix.

Model Development

The recommendation model calculates similarity using cosine similarity on book attributes, prioritizing genre, author, rating, and summary content. The content-based model ensures that recommendations are aligned with the user's past reading preferences, especially when genre alignment is crucial.

To further refine the recommendation process and mitigate the risk of over-recommending popular genres, I implemented a genre-weighted scoring system. This approach balances individual user preferences with genre diversity, allowing for a more personalized experience. Specifically, the model combines scores from various attributes—genre similarity, author similarity, rating similarity and content similarity—using defined weights. This hybrid method enhances the relevance of the recommendations while introducing users to a broader range of books, ultimately fostering a richer reading experience.

Website Development and Features

The website was built using Streamlit, designed with three main tabs:

  • All Books: Displays a Netflix-style grid of all available books, each showing the cover image, title, and author. Clicking on a book reveals its details.
  • My Books: A personalized library showcasing the books I have read.
  • Recommendations: Provides a curated list of book suggestions based on my reading history.

Results and Insights

The recommendation engine successfully provided relevant book suggestions, aligning with user (my) preferences. The system's efficiency was validated by user (my) feedback. Thanks to project I am currently reading books from the recommended by the system, enriching my reading experience.
Future iterations could incorporate collaborative filtering which would use data from other users to enhance the accuracy of recommendations or additionally, integrating Natural Language Processing (NLP) and sentiment analysis user reviews for enhanced accuracy and personalization.

Conclusion

This personalized recommendation system enhances the reading experience by providing curated suggestions aligned with the user’s interests.

What I Learned

This project provided insights into the intricacies of developing a book recommendation system. Here are some key takeaways:

  • Machine Learning Techniques: I became proficient in applying machine learning techniques, particularly cosine similarity and content-based filtering, to create an effective recommendation engine. This experience enhanced my ability to manipulate and analyze data to extract meaningful insights.
  • Hybrid Recommendation Systems: I learned about the advantages of combining different recommendation approaches. Integrating different content-based matrix can significantly improve the accuracy and relevance of recommendations.
  • Natural Language Processing: Exploring NLP techniques for text embedding provided me with a greater understanding of how to extract value from text.
